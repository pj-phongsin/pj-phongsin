## 🚀 Featured Project

### [StoreMind — AI Retail Intelligence Platform](https://github.com/pj-phongsin/storemind)
A full-stack retail management platform combining demand forecasting and AI workforce scheduling, inspired by real retail operations.

- 🧠 **ML Demand Forecasting** — Random Forest models trained on 1 year of sales data with Australian seasonal patterns
- 🤖 **AI Agent** — Autonomous sick leave handling & shift swap with P1/P2/P3 priority reallocation
- 📅 **DWS Generator** — Period-first daily work schedule algorithm with staggered breaks & TIDY periods
- 📊 **6 Dashboard Pages** — Sales, Inventory, Roster, Forecast, AI Agent, DWS Gantt chart
- 🐳 **Docker** — 4-container setup (MySQL + Node.js + Python FastAPI + nginx)

**Tech:** React • Vite • Tailwind CSS • Node.js • Express • Python • FastAPI • Scikit-Learn • MySQL • Docker
